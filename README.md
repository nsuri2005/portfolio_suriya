# Student Portfolio Website

A modern, elegant one-page portfolio website built with React.js showcasing a student's projects, skills, and contact information.

## Features

- **Sticky Navbar** with smooth scroll links
- **Hero Section** with split layout design
- **About Me** section with interests
- **Skills Section** with progress bars and skill chips
- **Projects Section** with project cards and GitHub links
- **Contact Section** with contact form and social links
- **Fully Responsive** design
- **Modern Design** with blue/purple accent colors
- **Smooth Animations** and hover effects

## Tech Stack

- **React.js** (Functional Components + Hooks)
- **Plain CSS** (No external frameworks)
- **No routing** (Single Page Application)

## Installation

1. Navigate to the project folder:
   ```bash
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

The application will run at `http://localhost:3000`

## Customization

- **Personal Information**: Edit component files to add your name, bio, and project details
- **Profile Image**: Replace the placeholder image URL in `Hero.js`
- **Color Scheme**: Modify CSS variables in `index.css` to change accent colors
- **Contact Links**: Update email and social links in `Contact.js`
